Graphical user interface

In this field of the setup file the user can set the configuration of the Graphical User Interface (GUI). The code below shows the section of the 'default_lokibc_setup.in' file that handles this configuration.

% --- configuration of the grafical user interface ---
gui:
  isOn: true
  refreshFrequency: 1

IMPORTANT NOTES:

  • The isOn parameter allows the user to activate (true) or deactivate (false) the GUI;
  • In refreshFrequency the user can control the frequency with which the GUI is refreshed. The value of this parameter corresponds to the number of required simulations for the GUI to be updated, hence the larger this value the less frequently the GUI is updated.
    Note that a very frequent update of the GUI can considerably degrade the run time. Note further that the GUI is optimized for Full HD screen resolution. Adopting different resolutions may result in anomalous displays.

When running LoKI-GM with the GUI activated (isOn: true), the graphic interface appears as in the following figure:

  • On the left-hand upper corner, for each run and depending on the selected tab, the GUI displays the Results (graphical) of the time evolution of the densities of species (in m−3 for volume densities and m−2 for surface densities) and the gas-related temperatures (Chemistry Vs Time), or the EEDF and the corresponding first anisotropy (EEDF), calculated for the self-consistent E/N value obtained from the simulations.
    The user can select/unselect a X axis logscale in the Chemistry Vs Time graphs, and/or a Y axis logscale in all the graphs (Chemistry Vs Time and EEDF).
    The user can select to display the results of a specic Run (rst tab), and/or the results for a specic gas (second tab in Chemistry Vs Time), and/or the results for a specic species (third tab in Chemistry Vs Time). Each of the previous selections adds one or more curves to the graphs (in addition to the gas-related temperatures in Chemistry Vs Time).
  • On the left-hand lower corner, for each run and depending on the selected tab, the GUI displays a summary of the Results (text) that can be found in the output files (see section 3.5), namely
    • in Final densities, quantitative information about the final values of the particle balance in the Chemistry, including the absolute density Nki, the population ξki and the relative error of the rate balance equation, for each species i of "gas" k, separating the Volume Chemistry and the Surface Chemistry; quantitative information about the population ξki of each species i of gas k used in the calculation of the Electron Kinetics, by solving the EBE. The information is organized by gases and their corresponding states (see Reference Manual [11]). The final percentage 100χk % of each gas in the mixture is indicated in square brackets;
    • in Final temperatures, quantitative information about the final values of the gas-related temperatures that are present in the workingConditions object;
    • in Chem Parameters, a summary of results relevant for the chemistry calculations, e.g. the final values of E/N, p, Tg, N, Qout, ne, I and/or P/V, and the corresponding relative errors (when applicable);
    • in Final balance, details about the particle balance for each species, as calculated by the Chemistry solver. Note that the same gain/loss rate may appear in different places with different values and units, if it relates to a reaction involving both volume and surface species;
    • in Power balance, details about the electron power balance, as calculated by the Boltzmann solver for the self-consistent E/N value obtained from the simulations;
    • in Swarm Parameters, a summary of the electron swarm parameters, calculated for the self-consistent E/N value obtained from the simulations;
    • in Rate Coefficients, a summary of the inelastic/superelastic electron rate coefficients, calculated for the self-consistent E/N value obtained from the simulations, using both the cross sections adopted for solving the EBE and the extra set of cross sections dened by the user as additional input data of LoKI-B (see section 3). It also displays the description of each mechanism.
  • On the right-hand upper corner, depending on the selected tab, the GUI displays a summary of the input Setup, namely
    • in setup file, the uncommented lines of the setup file;
    • in Rate Coefficients, an interactive list of the reactions included in the chemFiles. By clicking on each reaction, the GUI shows the main properties related with the reaction;
    • in Cross Sections, the graphs of the cross sections uploaded for the simulations (including both those adopted for solving the EBE and the extra cross sections dened by the user as additional input data of LoKI-B). Here, the user can select in the appropriated tab which cross section(s) to display.
  • On the right-hand lower corner, depending on the selected tab, the GUI displays a summary of the Status of the simulations, namely
    • in Simulation Log, information documenting events relevant to the code execution;
    • in Chemistry Iterations, graphical and quantitative information on the progress of the relative error of the simulations (identifying positive and negative error values), as a function of the iteration numbers, for the different nested calculation cycles (see Reference Manual, [11]), namely (from right to left), the Pressure cycle, the Quasineutrality cycle and the Global cycle.

When running LoKI-B as a standalone tool for electron kinetics stationary simulations (DC and HF cases) with the GUI activated (isOn: true), the graphic interface appears as in the following figure:

  • On the left-hand upper corner, for each value of E/N (or Tₑ) adopted in the simulations and depending on the selected tab, the GUI displays the Results (graphical) that can be found in the output files (see Reference Manual, [11]), namely
    • in EEDF, the plot of the EEDF and the corresponding first anisotropy;
    • in Electron Reduced Diffusion, the plot of DₑN as a function of E/N (or Tₑ);
    • in Electron Reduced Mobility, the plot of μₑN (and also ℜ[μ HFN] and −ℑ[ μHFN] in a HF case), as a function of E/N (or Tₑ);
    • in Electron Reduced Energy Diffusion, the plot of DεN as a function of E/N (or Tₑ);
    • in Electron Reduced Energy Mobility, the plot of μεN as a function of E/N (or Tₑ);
    • in Electron Energies, the plot of Tₑ ≡ (2/3)ε and uₖ, as a function of E/N (or Tₑ);
    • in Townsend Coefficient, the plot of α/N as a function of E/N (or Tₑ);
    • in Attachment Coefficient, the plot of η/N as a function of E/N (or Tₑ);
    • in Power Channels, the plot of the electron energy transferred in different channels, relative to the reference power, Θ x/Θ ref, as a function E/N (or Tₑ) (see Reference Manual, [11]).
    In all the graphs (except for EEDF and Power Channels), the user can select/unselect a X axis logscale and/or a Y axis logscale.
    The GUI supports graphs for multiple values of E/N (or Tₑ) and Tg. In ths case (i) each curve in the graph EEDF corresponds to a pair of values (E/N, Tg) or (Tₑ, Tg): (ii) all other graphs plot points/curves for the different Tg values considered.
    GUI support to multiple electron kinetics stationary simulations (with LoKI-B) is currently limited to simultaneous variations of these parameters.
  • On the left-hand lower corner, for each value of E/N (or Tₑ) adopted in the simulations and depending on the selected tab, the GUI displays a summary of the Results (text) that can be found in the output files (see section 5.4), namely
    • in Power balance, details about the electron power balance;
    • in Swarm Parameters, a summary of the electron swarm parameters (the Townsend coefficient, the attachment coefficient, and the drift velocity are only calculated and displayed in a DC case);
    • in Rate Coefficients, a summary of the inelastic/superelastic electron rate coecients, calculated using both the cross sections adopted for solving the EBE and the extra set of cross sections defined by the user as additional input data of LoKI-B (see section 3). It also displays the description of each mechanism
  • On the right-hand upper corner, depending on the selected tab, the GUI displays a summary of the input Setup, namely
    • in Setup file, the uncommented lines of the setup file;
    • in Cross Sections, the graphs of the cross sections uploaded for the simulations (including both those adopted for solving the EBE and the extra cross sections defined by the user as additional input data of LoKI-B). Here, the user can select in the appropriated tab which cross section(s) to display.
  • On the right-hand lower corner, the GUI will display a summary of the Status of the Simulation Log, with information documenting events relevant to the code execution.

When running LoKI-B as standalone tool for electron kinetics time-dependent simulations (e.g. considering a time-dependent, non-oscillatory (ω = 0) electric field) with the GUI activated (isOn: true), the graphic interface looks similar to the last figure. In this case

  • the Results (graphical) and the Results (text) are displayed as a function of the time t (instead of E/N or Tₑ );
  • the GUI displays also a Pulse temporal integration progress bar that depicts the progress of the simulation as a function of the Integration time, indicating also the current Computational time.

    • In this case, the GUI also supports graphs for multiple values of t and Tg, depicting: (i) curves for pairs of values (t, Tg) in the graph EEDF; (ii) points/curves for the different Tg values considered, in all other graphs.
    GUI support to multiple electron kinetics time-dependent simulations (with LoKI-B) is currently limited to simultaneous variations of E/N(t) and Tg.
